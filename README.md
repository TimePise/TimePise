## 📌 주요 프로젝트

### 🛡 낙상 감지 시스템 (캡스톤 디자인)
> **실시간 영상 분석 기반 고령자 낙상 감지 및 알림 시스템**
- 🧠 GRU 모델과 MediaPipe를 활용한 자세 인식 및 위험 판단
- 📡 Django 기반 실시간 WebSocket 알림 기능 구현
- 📹 OpenCV를 통한 스트리밍 및 낙상 발생 위치 정보 추출
- 📁 전체 시스템은 Django 서버 + OpenPose 기반 영상 분석 + UI 출력 구조로 설계됨
